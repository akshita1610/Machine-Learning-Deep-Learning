## 🚀 Quick Start

### Interactive Image Classification Tool (Recommended)
```bash
# Backend
cd Interactive-Image-Classification-Tool
python main_simple.py

# Frontend
# Open frontend_enhanced.html in your browser
# Navigate to http://localhost:8001
```

### Other Projects
```bash
# Image Classification
cd Image-Classification
python src/train.py

# Handwritten Digit Recognition
cd Handwritten-Digit-Recognition-Tool
python "1. K Nearest Neighbors/knn.py"
```
